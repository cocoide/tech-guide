### 機能一覧
- Youtube, Speakerdeck, Github, Zennなど様々なインターフェースを横断的に技術のキャチアップを可能にするタイムライン機能
→ 最新,トレンド, コメント数が多い, カスタムフィードの4つのビューが存在
- ストック機能 (公開範囲の設定も可能にする予定)
- 概要、要約機能
- 関連記事機能
- コントリビューション機能 (日々の活動がポイントで可視化)
